# Personal Portfolio Website

A modern and responsive portfolio website built to showcase my projects, skills, achievements, and development journey. This project also includes an admin dashboard for managing projects dynamically with authentication and image upload functionality.

---

## 🚀 Features

* Responsive and modern UI
* Admin authentication system
* Add, edit, and delete projects
* Upload project and profile images
* Dynamic content management
* MongoDB database integration
* Secure JWT authentication
* Contact section
* Skills showcase
* Clean and user-friendly interface

---

## 🛠️ Tech Stack

### Frontend

* Next.js
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### Additional Tools

* JWT Authentication
* Multer
* Local Storage

## ⚙️ Installation

### Clone the repository

```bash
git clone <your-repository-url>
```

### Install dependencies

```bash
npm install
```

### Setup environment variables

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the project

```bash
npm run dev
```
